# GitHub two-factor authentication

## 1. 介绍

登录 GitHub 官网，会提示要开启双因子认证。

但推荐的 APP 都是国外了，国内用不了。

可以使用 “腾讯身份验证器” 微信小程序。

## 2. 操作

开启双因子认证：

1. 打开 “腾讯身份验证器” 微信小程序，扫描 GitHub 那个二维码即可
2. 保存 “github-recovery-codes.txt” （这个暂时不知是干嘛的）
3. 完事

登录 GitHub：

1. 输入密码
2. 打开 “腾讯身份验证器” 微信小程序，查看 GitHub 对应的令牌，输入令牌
3. 完事
