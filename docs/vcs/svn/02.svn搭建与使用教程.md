# SVN 搭建与使用教程

>作者: 吴钦飞
>
>创建日期: 2025-08-21 09:54:20
>
>修改日期: 2025-08-21 11:26:57

## 1. 介绍

编撰此文档的目的是为说明如何使用 SVN。

概要:

* 安装 SVN server
* 打开 SVN server
* 创建仓库
* 创建 SVN 账号
* 安装 SVN 客户端
* 下载仓库
* 上传文件
* 更新仓库

## 2. 安装 SVN server

下载：

* [VisualSVN Server](https://www.visualsvn.com/downloads/)

安装:

![image-20250821112521007](./images/image-20250821112521007.png)

## 3. 打开 SVN server

启动 SVN Server

应用程序位置: `C:\Program Files\VisualSVN Server\bin\VisualSVN Server.msc`

## 4. 创建仓库

![image-20250821105044846](./images/image-20250821105044846.png)

## 5. 创建 SVN 账号

创建账户 test/123456

![image-20250821100636397](./images/image-20250821100636397.png)

## 6. 安装 SVN 客户端

下载:

* [TortoiseSVN-1.14.5.29465-x64-svn-1.14.2.msi](https://webcdn.m.qq.com/spcmgr/download/TortoiseSVN-1.14.5.29465-x64-svn-1.14.2.msi)


安装: 

* 点击 “next” 即可
* 可以更改安装路径

## 7. 下载仓库

第 1 步: 在本地创建存放仓库的目录，如下

`D:\dev\labwind\svn`

第 2 步: 在 目录 空白处 右键菜单

![image-20250821102403951](./images/image-20250821102403951.png)

第 3 步: 获取仓库地址 `http://114.215.175.167:8089/svn/WindSheetDeployment/`

![image-20250821103804294](./images/image-20250821103804294.png)

第 4 步: 输入仓库地址，进行下载

![image-20250821102825872](./images/image-20250821102825872.png)


注意事项:

* 前端编译后代码（如 `dist` 文件夹），请转为压缩文件（`dist.zip`）再进行上传

## 8. 上传文件

仅用于测试
