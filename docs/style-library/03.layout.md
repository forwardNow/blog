# 布局

归纳总结常见布局

## 1. 居中

### 1.1. 垂直居中

#### 文本/行内元素/行内块级元素

示例：

<div class="bd-example">
  <div class="text-center">我是一段文本内容</div>
  <div class="text-center mt-2"><button>按钮</button></div>
</div>

代码：

```css
.text-center {
  text-align: center !important;
}
```

说明：

* `text-align` 只控制行内内容(文字、行内元素、行内块级元素)相对其父级元素的对齐方式

注意：

* 设置父元素的属性
* 只对行内内容有效
* `text-align` 属性会被继承


#### 单个块级元素

示例：

<div class="bd-example">
  <div class="margin-x-auto width-25 border">定宽（25%） div</div>
</div>

代码：

```css
.margin-x-auto {
  margin-right: auto !important;
  margin-left: auto !important;
}

.width-25 {
  width: 25% !important;
}
```

原理：

* `margin-left: auto; margin-right: auto;`, 会均分剩余的空间
* `margin-top: auto; margin-bottom: auto;`, 计算为 0
* 左右单个为 `auto` 会沾满剩余空间

注意：

* 必须定宽，并且值不能为 `auto`
* 宽度要小于父元素，否则无效
