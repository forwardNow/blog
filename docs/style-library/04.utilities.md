# 工具类

[[toc]]

## 1. 说明

基于 [bootstrap 5.1](https://v5.bootcss.com/docs/utilities/background/) 的 utilities 和 helpers 改造

内容跟官网的基本一致，不同的地方：

* 去掉了响应式相关的 CSS 类
* 扩展（新增而非修改）了一些类名，方便使用，如：
  * `mb` --> `margin-bottom`
  * `xx-start` --> `xx-left`
  * `xx-end` --> `xx-right`

## 5. 元素类型（display）

语法：`d-{value}`

`{value}` 的值为：
* `inline` 
* `inline-block` 
* `block` 
* `grid` 
* `table` 
* `table-row` 
* `table-cell` 
* `flex` 
* `inline-flex` 
* `none`

示例：

<div class="bd-example">
  <div class="d-flex gap-2">
    <div class="p-5 bg-primary"></div>
    <div class="p-5 bg-info"></div>
    <div class="p-5 bg-danger"></div>
    <div class="p-5 bg-warning">
      上山打
      <span class="d-none">老虎</span>
    </div>
  </div>
</div>

代码：

```html
<div class="d-flex gap-2">
  <div class="p-5 bg-primary"></div>
  <div class="p-5 bg-info"></div>
  <div class="p-5 bg-danger"></div>
  <div class="p-5 bg-warning">
    上山打<span class="d-none">老虎</span>
  </div>
</div>
```

CSS 类：

```css
.d-inline {
```
