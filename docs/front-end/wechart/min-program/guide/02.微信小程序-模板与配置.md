## 1. WXML 模板语法

### 1.1. 数据绑定

基本原则：

1. 在 data 中定义数据
2. 在 WXML 中使用数据

在 data 中定义页面的数据：

```js
Page({
  data: {  
    person: { name: '张三', age: 18 }
  }
})
```

mustache 语法：

* 双花括号，插值表达式
* 示例:
  * index.wxml
    
    ```html
    <view>{{ message }}</view>
    ```

  * index.js

    ```js
    Page({
      data: {
        message: 'hello world'
      },
    })
    ```
