# console - 控制台打印

## 1. 介绍

`console.log()` 控制台打印

## 2. 分组

### 2.1. 展开内容的组

代码：

```js
console.group('标题-展开内容的组')
console.log('内容1')
console.groupEnd()
```

图示：

![image-20240815101144750](images/image-20240815101144750.png)

### 2.2. 折叠内容的组

代码：

```js
console.groupCollapsed('标题-折叠内容的组')
console.log('内容1')
console.groupEnd()
```

图示：

![image-20240815101232176](images/image-20240815101232176.png)
