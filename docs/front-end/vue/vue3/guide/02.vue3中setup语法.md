# vue3 中 setup 语法

## 1. 说明

在 `<script setup>` 中定义的顶级变量都可以直接在模板中使用

## 2. props

```html
<script setup lang="ts">
  export interface Props {
    name?: string,
    hobby?: string[],
  }

  const props = withDefaults(defineProps<Props>(), {
    name: 'light',
    hobby: ['打乒乓球', '打篮球'],
  });
</script>
```

## 3. model

```html
<script setup>
  const props = defineProps({
    name: String,
    age: Number
  })

  const emit = defineEmits(['update:name', 'update:age']);

  emit('update:name', '李四');
  emit('update:age', 19);
</script>
```
