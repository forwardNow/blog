# vue3 中 vue-router 路由缓存 keepAlive

## 示例

```vue
<template>
  <router-view v-slot="{ Component }">
    <component :is="Component" v-if="!$route.meta.keepAlive" />
    <keep-alive>
      <component :is="Component" v-if="$route.meta.keepAlive" />
    </keep-alive>
  </router-view>
</template>
```
