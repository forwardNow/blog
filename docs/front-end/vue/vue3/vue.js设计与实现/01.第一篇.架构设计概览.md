# 第一篇.架构设计概览

## 1. 第 1 章 权衡的艺术

### 1.1. 命令式和声明式

视图层框架按范式分为：

* 命令式：关注过程，与自然语音一一对应
* 声明式：关注结果，由框架来实现过程

示例：

```js
// 自然语言
//  1. 获取元素： id 为 “app” 的 div 元素
//  2. 设置文本：内容为 “hello world”
//  3. 绑定事件：点击事件
//  4. 点击后弹出 ok

// jQuery
$('#app')
  .text('hello world')
  .on('click', () => alert('ok'))

// vue
<div @click="() => alert('ok')">hello world</div>
```
