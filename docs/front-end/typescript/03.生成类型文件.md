# 生成类型文件

## 1. webpack 打包时生成类型文件

配置：

* tsconfig.json:

    ```json
    {
      "compilerOptions": {
        "declaration": true,
        "emitDeclarationOnly": false,
        "outDir": "dist",
        "declarationMap": true,
      },
    }
    ```

参考：

* [Creating .d.ts Files from .js files](https://www.typescriptlang.org/docs/handbook/declaration-files/dts-from-js.html)
