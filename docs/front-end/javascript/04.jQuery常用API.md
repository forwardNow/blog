# jQuery 常用 API

## 1. 介绍

jQuery 常用 API

## 2. 位置

### 2.1. .offset()

说明：

* 获取文档位置，相对文档

示例：

```js
$('.box').offset();
//=> { left: 100, top: 200 }

$('.box').offset({ left: 100, top: 200 });
```

### 2.2. .position()

说明：

* 相对父元素的位置

示例：

```js
$('.box').position();
//=> { left: 100, top: 200 }
```
