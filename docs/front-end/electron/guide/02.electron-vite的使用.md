# electron-vite 的使用

## 介绍

官网: https://cn-evite.netlify.app/

## 安装

使用 node v20 的版本

```shell
# 这个命令可以直接创建项目
npm create @quick-start/electron@latest
```

## 路径

### resources

```text
${root}/
  resources/
    hello.exe
  src/
    main/
      index.js
```

```js
// 在 index.js 获取 hello.exe 的路径
path.join(__dirname, "../../resources/hello.exe").replace("app.asar", "app.asar.unpacked");
```
