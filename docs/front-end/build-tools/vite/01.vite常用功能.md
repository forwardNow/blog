# vite 常用功能

## 环境变量

文件:

```text
${root}/
  .env
  .env.development    # "dev": "vite",
  .env.production     # "build": "vue-tsc -b && vite build",
```

.env.development:

```text
VITE_API_URL=http://localhost:8088/labwind-admin
```

.env.production:

```text
VITE_API_URL=//labwind-admin
```

在业务代码里使用:

```js
console.log(import.meta.env.VITE_API_URL);
```
