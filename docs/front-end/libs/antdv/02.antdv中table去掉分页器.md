# antdv 中 table 去掉分页器

## 1. 介绍

ant-design-vue (4.1.2) 中的 table 组件，去掉默认的分页器。

## 2. 示例

```vue
<template>
  <a-table
    :pagination="false"
  >
  </a-table>
</template>
```
