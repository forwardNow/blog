# el与data的两种写法

## 挂载元素

```html
<div id="app"></div>
```

方式一：

```javascript
new Vue({
  el: '#app',
});
```


方式二：

```javascript
// 更灵活
new Vue({}).$mount('#app');
```

## 数据集

方式一：（对象式）

```javascript
new Vue({
  data: {
    name: 'zhangsan,'
  },
});
```

方式二：（函数式）

```javascript
new Vue({
  // 由 Vue 管理的函数，不要使用箭头函数
  // 如果使用箭头函数， this 为 window
  data() {
    return {
      name: 'zhangsan,'
    }
  },
});
```
