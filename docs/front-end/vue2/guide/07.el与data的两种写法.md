# el与data的两种写法

## 挂载元素

```html
<div id="app"></div>
```

方式一：

```javascript
new Vue({
  el: '#app',
});
```


方式二：

```javascript
// 更灵活
new Vue({}).$mount('#app');
```

## 数据集

方式一：

```javascript
new Vue({
  data: {
    name: 'zhangsan,'
  },
});
```

方式二：

```javascript
new Vue({
  data() {
    return {
      name: 'zhangsan,'
    }
  },
});
```
