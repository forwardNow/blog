# AES 前端加解密

## 1. 介绍

AES, Advanced Encryption Standard, 高级加密标准

对称加密, 对明文的加密和解密需要使用同一个密钥

AES支持三种长度的密钥：

* 128位, AES128
* 192位, AES192
* 256位, AES256
