# 解决访问github慢的问题

## 原因

GitHub 的 CDN 域名遭到 DNS 污染，无法连接使用 GitHub 的加速分发服务器，所以国内访问速度较慢。

也就是说 DNS 出问题了，那就改 hosts 文件。

## 使用

1. 获取 github hosts 文件：

   ```text

   https://raw.hellogithub.com/hosts
   ```
