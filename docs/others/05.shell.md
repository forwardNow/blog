## 1. 变量

### 1.1. 局部变量

说明：

* 在当前 shell 中生效

示例：

```shell
student_name='zhangsan'

echo $student_name
```


### 1.2. 全局变量

说明：

* 在所有 shell 脚本中生效

设置变量：

```shell
# var.sh
export hui_dir="/d/dev/projects/frontend"
```

使用环境变量：

```shell
# biz.sh
source ./vars.sh

$hui_dir
```

## 2. for

```shell
students=(zhangsan lisi wangwu)

for student_name in ${students[@]}
do
    echo '我的同学：'
    echo $student_name
done
```
