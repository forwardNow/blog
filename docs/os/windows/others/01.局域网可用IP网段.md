# 局域网 IP 网段

## 1. 局域网 IP 地址段

三大类：

* A类：`10` 段，后三位自由分配，也就是 `10.0.0.0` - `10.255.255.255`；
* B类：`172.16` 段，后两位自由分配，也就是 `172.16.0.0` - `172.31.255.255`；
* C类：`192.168` 段，后两位自由分配，也就是 `192.168.0.0` - `192.168.255.255`；

## 2. 特殊的 IP 地址

`127.0.0.1`:

* 本地回路测试地址

`255.255.255.255`:

* 广播地址

`0.0.0.0`:

* 任何网络
