# windows 更改 CPU 电源设置

## 介绍

有效延长 CPU 的寿命

## 步骤

步骤 1. 修改注册表，让电源选项暴露出来

`Win + R` 输入 `regedit` 打开注册表编辑器，编辑如下内容

```text
计算机\HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Power\PowerSettings\54533251-82be-4824-96c1-47b60b740d00\75b0ae3f-bce0-45a7-8c89-c9611c25e100
Attributes 的数据数据从 1 改为 2

计算机\HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Power\PowerSettings\54533251-82be-4824-96c1-47b60b740d00\be337238-0d82-4146-a960-4f3749d470c7
Attributes 的数据数据从 1 改为 2
```

步骤 2. 搜索 `编辑电源计划`

1. 点击 “更改高级电源设置”
2. 点击 “处理器电源管理”
3. 点击 “处理器性能提升模式”
4. 将 使用电池 和 接通电源 两个选项匀设置为“高效、积极且有保障”
