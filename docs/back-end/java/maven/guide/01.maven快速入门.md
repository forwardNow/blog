# maven 快速入门

## 1. 介绍

### 1.1. 说明

Maven 是 apache（开源软件基金会） 旗下的开源项目

用于管理和构建 Java 项目的工具。

### 1.2. 作用

依赖管理：

* 管理项目依赖的 jar 包，避免冲突

统一项目结构：

* 提供标准、统一的项目结构
* eclipse、MyEclipse、Idea 创建的 Maven 项目都可以互相导入
* ![统一项目结构](./images/01_maven_features_2.jpg)

项目构建：

* 自动化项目构建方式
