# MinIO

## 1. 介绍

说明：

* 对象存储（Object Storage）
* 开源GNU AGPL v3 （完全免费）

官网：

* https://min.io/
* http://minio.org.cn/


## 2. windows 上安装与启动

下载:

* https://www.minio.org.cn/download.shtml

目录:

```text
D:\download\minio
  data/
  minio.exe
```

启动:

```text
D:\download\minio> minio.exe server ./data --console-address ":9001"
```
