# springboot-tomcat-http请求大小设置

## springboot2

```yml
spring:
  servlet:
    multipart:
      # 设置整个请求的最大大小，包括所有文件和表单字段的总和。默认值：10MB
      max-request-size: 200MB
      # 设置单个文件的最大上传大小。默认值：1MB
      max-file-size: 100MB
      enabled: true

server:
  tomcat:
    # 限制 HTTP 表单提交（application/x-www-form-urlencoded）的最大 POST 请求大小。默认值：2MB
    # 这个值对 multipart/form-data 也有效
    max-http-form-post-size: 100MB
```
