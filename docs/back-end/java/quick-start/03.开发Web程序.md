#### 3.3.2. 修改 idea 内置 maven 配置：

```xml
<!-- 
  IntelliJ IDEA 2022.2.3\plugins\maven\lib\maven3\conf\settings.xml
-->

<mirror>
  <id>central</id>
  <mirrorOf>central</mirrorOf>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```
