# shiro

## 1. 入门概述

### 1.1. 是什么

Apache Shiro 是 Java 安全（权限）框架。

功能:

* 认证
* 授权
* 加密
* 会话管理
* 与 Web 集成
* 缓存

### 1.2. 为什么要用 Shiro

* 易于使用
* 全面
* 灵活
* 强力支持 Web
* 兼容性强

### 1.3. 与 Spring Security 的对比

* Spring Security 基于 Spring，更方便、强大，社区资源也更好
* Shiro 不依赖 Spring，在集群中 会话 独立于容器
