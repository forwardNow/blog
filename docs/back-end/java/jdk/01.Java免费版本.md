## 免费版本

Java 8: 8u202 是最后的免费版本

JDK11，免费版本到 11.0.2，从 11.0.3 版本开始商用收费。

JDK12、JDK13、JDK14、JDK15、JDK16，全版本商用收费。

JDK17、JDK18、JDK19、JDK20，全版本(二进制版本)免费。
