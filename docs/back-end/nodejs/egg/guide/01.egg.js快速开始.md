# egg.js 快速开始

## 1. 介绍

说明：

* 为企业级框架和应用而生的 node.js 框架
* 约定优于配置

特点：

* 基于 Koa

## 2. hello world

快速生成项目:

```shell
mkdir egg-example

cd egg-example

npm init egg --type=simple

npm i

# ? Please select a boilerplate type simple - Simple egg app boilerplate
```

启动项目:

```shell
# 修改源码后，自动“重启”
npm run dev

open http://localhost:7001
```
