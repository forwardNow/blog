# 限制项目的 node 版本

## 1. 介绍

使用 `npm install` 安装项目依赖时，限制 node 的版本

## 2. 使用

package.json:

```json
{
  "engines": {
    "node": ">= 14.15.0"
  }
}
```

.npmrc:

```text
engine-strict = true
```
