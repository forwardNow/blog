# MySQL-水平分表

## 介绍

在 MySQL 中，拆分（拆表）通常是指将一个大表分解成多个小表，以提高查询效率、减少单个表的数据量、优化索引等。

常见的拆表策略：

1. 垂直拆分，将表的列拆分到多个表中
2. 水平拆分，将表的行拆分到多个结构相同的表中
