# MySQL - 备份与恢复

## 备份

备份 数据库(`windsheet2024`) 到 `D:\\backup.sql`

包含 删库、建库 的语句

```shell
"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" --host=localhost --port=3306 --user=root --password=123456 --default-character-set=utf8 --protocol=tcp --column-statistics=FALSE --set-gtid-purged=OFF --lock-tables=FALSE --single-transaction=TRUE --routines --events --add-drop-database --databases windsheet2024 > "D:\\backup.sql"
```

## 恢复

```shell
"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" --protocol=tcp --host=localhost --port=3306 --user=root --password=123456  --default-character-set=utf8 --comments  < "D:\\backup.sql"
```
