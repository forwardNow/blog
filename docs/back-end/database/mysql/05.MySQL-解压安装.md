# MySQL-解压安装

## 初始化

解压 mysql-8.0.46-winx64.zip 到 D:\testmysql 

切换到 bin 目录执行 `mysqld --initialize --console`

会生成 data 目录及 默认密码

```text
D:\testmysql\mysql-8.0.46-winx64\bin>mysqld --initialize --console


2026-04-27T12:46:57.013151Z 0 [System] [MY-013169] [Server] D:\testmysql\mysql-8.0.46-winx64\bin\mysqld.exe (mysqld 8.0.46) initializing of server in progress as process 2624
2026-04-27T12:46:57.027160Z 1 [System] [MY-013576] [InnoDB] InnoDB initialization has started.
2026-04-27T12:46:57.159278Z 1 [System] [MY-013577] [InnoDB] InnoDB initialization has ended.
2026-04-27T12:46:57.761302Z 6 [Note] [MY-010454] [Server] A temporary password is generated for root@localhost: <gh/%);q%5-Q
```



mysqld --initialize-insecure

--initialize‌：这个选项会在初始化时生成一个随机密码，并将其输出到错误日志中（通常是 hostname.err 文件）。

--initialize-insecure‌：这个选项会初始化数据库但不设置密码，这意味着你可以无需密码登录。

### 注册并启动服务

必须使用管理员权限打开的 CMD

```text
D:\testmysql\mysql-8.0.46-winx64\bin>mysqld --install MySQL8046
Service successfully installed.

D:\testmysql\mysql-8.0.46-winx64\bin>net start MySQL8046
MySQL8046 服务正在启动 .
MySQL8046 服务已经启动成功。
```

### 修改密码

必须修改默认密码，不然 客户端工具（如 workbench）登录失败

```text
D:\testmysql\mysql-8.0.46-winx64\bin>mysqladmin -uroot -p"<gh/%);q%5-Q" password 123456
```

### 登录

```text
D:\testmysql\mysql-8.0.46-winx64\bin>mysql -uroot -p123456
```
