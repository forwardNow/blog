# chrome常用插件

## 插件

| 名称             | 说明                                         |
|----------------|--------------------------------------------|
| FeHelper(前端助手) | JSON自动格式化、手动格式化，支持排序、解码、下载等，更多功能可在配置页按需安装！ |
| [Chrome-Charset](https://github.com/jinliming2/Chrome-Charset/releases) | 指定页面编码，解决控制台 network 面板中请求预览乱码问题 |
