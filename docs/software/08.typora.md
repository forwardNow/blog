# typora

## 1. v1.0.3

步骤：

1. 安装 typora
2. 将 `app.asar.txt` 文件名中的 `.txt` 去掉，并覆盖至安装路径的“resources”文件夹内
3. 打开 typora ，点击“输入序列号”
4. 邮箱一栏中任意填写（但须保证邮箱地址格式正确）
5. 输入序列号，点击“激活”。
