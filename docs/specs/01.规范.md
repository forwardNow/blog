## 3. 前后端接口规范

### 3.1. 数据交换

* 请求

  * 请求方法： POST

  * 请求体编码： application/json
  
* 响应

  * 响应体编码： application/json

  * 响应体格式：

      ```json
      {
        "code": 200,
        "msg": "ok",
        "data": {}
      }
      ```

### 3.2. 分页

* 请求体

* 响应体

### 3.3. 非分页

* 请求体

* 响应体

### 3.4. 状态码

>业务处理结果的说明

通用状态码：

| 值（`code`） | 说明（`msg`） |
| - | - |
| 200 | Ok |
| 401 | Unauthorized |
| 500 | Server internal error |
